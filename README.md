Adds Interactive In-Store Canvas Map
Implements a 2D canvas map for in-store navigation that converts real-world GPS coordinates into a relative pixel layout. Includes custom pointer event handling for smooth panning.

How to Run & Test (Desktop & Mobile)

1. Install dependencies
Make sure you are in the root directory of your project in the terminal, then run: "npm install"

3. Start the development server
To test on your phone, you need to expose the local server to your Wi-Fi network: "npm run dev -- --host"

4. Test on your computer
Click the local link in your terminal, or open your browser to:
http://localhost:5173

5. Test on your phone

Ensure your phone and computer are connected to the same Wi-Fi network.

Look at your terminal output. You should see a Network URL listed.

Type that exact Network URL into your phone's browser .
